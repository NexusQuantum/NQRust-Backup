python-nqrustbackup Module
====================

`python-nqrustbackup` is a Python module to access a NQRustBackup backup system.

Packages for `python-nqrustbackup` are included in the NQRustBackup core distribution and available via https://pypi.org/project/python-nqrustbackup/.

The main functionality is to access a |dir| via its console interface.
This is the same interface as used by the :command:`nqrustbackup_console` command line tool
or the |webui|.

This module is not intended for NQRustBackup Python plugins. NQRustBackup plugins run inside the NQRustBackup daemons, while this module connects to the NQRustBackup Daemon control interfaces.

The relevant classes are:

  * :py:class:`nqrustbackup.bsock.directorconsole.DirectorConsole`
  * :py:class:`nqrustbackup.bsock.directorconsolejson.DirectorConsoleJson`

Example:

.. code:: python

   >>> import nqrustbackup
   >>> directorconsole=nqrustbackup.bsock.DirectorConsole(address='localhost', port=9101, name='user1', password='secret')
   >>> print(directorconsole.call('help').decode("utf-8"))

If you want to create a command line tool, the examples in
https://github.com/nqrustbackup/nqrustbackup/tree/master/python-nqrustbackup/bin might be useful.
