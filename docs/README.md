nqrustbackup-docs
===========


This repository contains the main documentation for the NQRustBackup project https://www.nqrustbackup.org/.

The documentation is published as HTML at https://docs.nqrustbackup.org.

The documentation source is done using RST.

If you like to contribute, please have a look at https://www.nqrustbackup.org/en/howto-contribute.html
