python-nqrustbackup
=============

`python-nqrustbackup` is a Python module to access a https://www.nqrustbackup.com backup system.

Packages for `python-nqrustbackup` are included in the NQRustBackup core distribution and available via https://pypi.org/.

Documentation is available at https://docs.nqrustbackup.org/DeveloperGuide/PythonNQRustBackup.html
